# Beer E-Commerce

A mobile-first e-commerce app for browsing and purchasing beer products. Built with React on the frontend and Express on the backend.

## Project Structure

- **frontend/** - React app with product listings and details
- **backend/** - Express API serving product and stock data

## Quick Start

### Backend
```bash
cd backend
npm install
npm run dev
```
Server runs on http://localhost:3001

### Frontend
```bash
cd frontend
npm install
npm run dev
```
App runs on http://localhost:3000

## Features

- Responsive product grid that adapts to mobile, tablet, and desktop
- Real-time stock and price updates
- Smooth sliding product detail panel
- Clean, modern UI with custom styling

## Tech Stack

- React + Vite
- Express.js
- SASS for styling
- React Router for navigation
